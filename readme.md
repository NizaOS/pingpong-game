<h1>Winter Ping-Pong</h1>
<p>A winter-themed 2D Pong game built with Python and Pygame.</p>
<hr>
<h2>How to Play</h2>
<p>You do not need to install Python or any external libraries to run this version of the game.</p>

<h3>Only For macOS Users for now 😭</h3>
<ul>
<li>Locate the <b>pingpong.app</b> (or the unzipped application).</li>
<li>Right-click the app and select <b>Open</b>.</li>
<li><i>Note: If macOS blocks the app, navigate to System Settings > Privacy & Security and click "Open Anyway" at the bottom of the page.</i></li>
</ul>
<hr>
<h2>Controls</h2>
<ul>
<li><b>UP Arrow</b>: Move paddle up</li>
<li><b>DOWN Arrow</b>: Move paddle down</li>
<li><b>Goal</b>: Prevent the ball from passing your paddle and score against the AI.</li>
</ul>
<hr>
<h2>Development and Source Code</h2>
<p>To run or edit the source code (pingpong.py), the following requirements must be met:</p>
<ol>
<li>Python 3.x installed.</li>
<li>Pygame library installed via pip:
<pre>pip install pygame</pre>
</li>
<li>Asset Files: Ensure <b>brick.png</b> and <b>background.jpg</b> are located in the same directory as the script.</li>
</ol>
