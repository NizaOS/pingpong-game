# Winter Ping-Pong

This is a 2D Pong game built with Python and Pygame.

## Installation

You must have pygame installed to run this script. Install it using pip:

pip install pygame

## How to Play

Run the game with:

python3 pingpong.py

Use the UP and DOWN arrow keys to control the paddle.

## Requirements

The following files must be in the same folder:
1. pingpong.py
2. brick.png
3. background.jpg
